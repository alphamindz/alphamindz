## 🚀 Featured Projects

### 🎓 [Skill-Lab](https://github.com/alphamindz/skill-lab) — E-Learning Platform
> **Tech:** Next.js 14 App Router · Prisma ORM · Server Actions · PostgreSQL · TypeScript

A full-stack, production-grade e-learning platform engineered for **scale and performance**. Features include course management, real-time progress tracking, and a seamless learner dashboard. Built with the latest Next.js App Router paradigm, emphasizing RSC (React Server Components), optimistic UI updates, and edge-ready deployments.

- ✅ Server Actions for zero-boilerplate mutations
- ✅ Role-based access control (Student / Instructor / Admin)
- ✅ Prisma ORM with PostgreSQL for relational data integrity

### 🧠 [Kmc-Algo](https://huggingface.co/alphamindz) — RL Training Environment
> **Tech:** Python · PyTorch · Gymnasium · Hugging Face Spaces · Reinforcement Learning

A specialized **Reinforcement Learning environment** designed for training agents on algorithmic tasks. Agents learn to sort, search, and traverse data structures through reward-driven exploration — bridging classical CS with modern deep RL.

- ✅ Custom Gymnasium-compatible environments for algorithm simulation
- ✅ Trained and deployed models on Hugging Face Spaces
- ✅ PPO & DQN agent implementations from scratch

### 🤝 [StudentFund](https://github.com/alphamindz/studentfund) — Open Source Crowdfunding for Students
> **Tech:** Next.js · MongoDB · Express.js · Node.js

A **collaborative open-source ecosystem** enabling students to crowdfund their projects, research, and startup ideas. Designed as a peer-to-peer platform with transparent fund flows and milestone-based funding releases.

- ✅ Milestone-based disbursement with smart fund locking
- ✅ Public project discovery feed with search & filter
